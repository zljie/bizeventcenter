# 企业中间件业务事件中心POC前端展示项目

## 项目概述

这是一个基于React + TypeScript + Ant Design X5的商务风格企业级后台管理系统POC（概念验证）项目，专注于企业中间件业务事件管理，特别针对SAP ERP石油化工行业业务场景。

## 技术栈

- **前端框架**: React 18.3 + TypeScript
- **UI组件库**: Ant Design 6.0
- **构建工具**: Vite 6.0
- **路由管理**: React Router v6
- **样式方案**: Tailwind CSS + Ant Design主题

## 核心功能模块

### 1. 仪表板（Dashboard）
- 实时统计数据展示（已发布事件、总订阅数、今日触发、成功率）
- 最近事件日志列表
- 活跃事件概览

### 2. 消息主体登记（事件注册中心）
- **SAP ERP石油化工事件模板库**（19个专业模板）
- 事件分类体系：生产事件、采购事件、销售事件、库存事件、安全事件
- 事件属性配置：事件ID、名称、业务域、触发条件、数据结构、优先级
- 搜索和筛选功能
- 查看详情弹窗

### 3. 事件发布中心
- 事件列表管理（表格展示）
- 新增/编辑/删除事件功能
- **批量操作功能**：
  - 批量选择（checkbox）
  - 批量发布
  - 批量停用
  - 批量删除
- 状态管理（草稿、已发布、已停用）
- 搜索和分类筛选

### 4. 事件订阅中心
- 订阅关系展示和管理
- 订阅统计（正常/暂停/成功推送/失败推送）
- 新增订阅配置
- 多种推送方式支持（Webhook、HTTP API、消息队列、邮件）

### 5. 业务事件日志
- 详细的事件日志记录
- 日志统计数据
- 搜索、状态筛选、日期范围筛选
- 查看详情功能（包含事件载荷Payload）

### 6. 系统设置
- 系统配置管理
- 事件处理参数配置（超时时间、重试次数、重试间隔）
- 日志和通知配置

## UI/UX特性

- 可折叠侧边栏（71%空间节省，220px ↔ 64px）
- 简化版面包屑导航（显示当前页面标题）
- 商务风格配色（蓝色主色调 #1890ff）
- 响应式布局
- 现代化表格（排序、筛选、分页）
- 批量操作（多选、批量处理）
- 图标系统（Ant Design Icons）

## ⚠️ 重要说明：数据持久化

### 当前实现（POC阶段）

**本项目是POC（概念验证）前端展示项目，使用前端模拟数据：**

1. **数据来源**：
   - 所有数据存储在 `public/data/` 目录下的JSON文件中
   - 包括：事件模板、已发布事件、订阅关系、事件日志

2. **数据操作特性**：
   - ✅ 新增/编辑/删除操作仅在前端生效
   - ⚠️ 页面刷新后会重置为初始数据
   - ⚠️ 不同浏览器/设备间数据不同步
   - ℹ️ 这是POC项目的预期行为，用于功能演示和原型验证

3. **适用场景**：
   - 功能演示和概念验证
   - UI/UX设计评审
   - 用户体验测试
   - 业务流程确认

### 生产环境升级方案

如需将此项目升级为生产级应用，需要进行以下改造：

1. **后端API集成**：替换fetch调用为真实API
2. **状态管理优化**：引入Redux或Zustand
3. **认证授权**：集成企业SSO或OAuth2.0
4. **数据持久化**：后端数据库（PostgreSQL或MongoDB）
5. **实时功能**：WebSocket或Server-Sent Events

## 快速开始

### 安装依赖
```bash
pnpm install
```

### 开发模式
```bash
pnpm run dev
```
访问 http://localhost:5173

### 生产构建
```bash
pnpm run build
```
构建产物位于 `dist/` 目录

- 10个已发布事件
- 8个订阅关系
